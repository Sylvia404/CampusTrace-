##  Project Overview

**CampusTrace** is a comprehensive web-based Lost and Found Management System developed specifically for the **Institute of Accountancy Arusha (IAA)**. It streamlines the entire process of reporting, searching, claiming, and returning lost items on campus.

### Why CampusTrace?

| Problem | Solution |
|---------|----------|
| Lost items go unreported | Easy posting system |
|  Hard to find lost belongings | Advanced search with filters |
|  No proof of ownership | Claim system with verification |
|  No communication channel | Secure messaging system |
|  No oversight | Admin dashboard with analytics |
|  No transparency | Real-time status tracking |

##  Features

###  For Students

<details>
<summary><b> Dashboard</b></summary>

- Statistics overview (Total items, Open items, Claims)
- Quick action buttons
- Recent activity feed
- Item breakdown charts
- Claim status monitoring
</details>

<details>
<summary><b> Item Management</b></summary>

- Post found items with images
- Post lost items with description
- View all personal items
- Filter by type (Found/Lost)
- Filter by status (Open/Claimed/Returned)
- Edit item details
- Delete items
- Upload item images
</details>

<details>
<summary><b> Search & Discovery</b></summary>

- Keyword search
- Category filtering
- Location filtering
- Status filtering
- Sort by newest/oldest/popular
- View item details
</details>

<details>
<summary><b> Claim System</b></summary>

- Submit claims with proof
- Provide detailed descriptions
- Upload proof images
- Track claim status
- Owner approval/rejection
- Claim history
</details>

<details>
<summary><b>💬 Messaging System</b></summary>

- Chat enabled after claim approval
- Private conversations
- Unread message indicators
- Conversation history
- Real-time messaging
</details>

<details>
<summary><b> Notifications</b></summary>

- Claim status updates
- New message alerts
- Meeting reminders
- System announcements
</details>

###  For Administrators

<details>
<summary><b>Analytics Dashboard</b></summary>

- Total users count
- Item statistics
- Claim statistics
- Monthly activity charts
- Item distribution charts
- Status breakdown
- Recent activity feed
</details>

<details>
<summary><b> User Management</b></summary>

- View all users
- Search users
- Activate/Suspend accounts
- Delete users
- View user details
</details>

<details>
<summary><b> Item Moderation</b></summary>

- View all items
- Filter by type/status
- Delete spam/offensive items
- View item details
</details>

<details>
<summary><b> Claim Resolution</b></summary>

- View all claims
- Filter by status
- Approve/Reject claims
- View proof details
- Add admin notes
</details>
